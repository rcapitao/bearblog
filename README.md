# bear-plugins
Coleção de plugins de terceiros utilizados no meu blog, hospedado no [Bear Blog](https://bearblog.dev).
Este repositório serve como catálogo e referência dos plugins instalados, facilitando o controle de versões, dependências e reinstalação quando necessário.

## Sobre
Os plugins do Bear Blog estendem funcionalidades do blog via snippets de CSS, JavaScript e HTML inseridos nos campos de personalização. Aqui mantenho registro dos plugins de terceiros que utilizo, com links para as fontes originais e notas sobre configuração.

## Instalação
No Bear Blog, os plugins são adicionados via snippets de código nos campos de personalização do painel:

1. Acesse **Dashboard → Settings**.
2. Cole o snippet de CSS no campo **Custom CSS/Style**.
3. Cole snippets de HTML/JavaScript no campo **Header** ou **Footer** (script tags), conforme a documentação de cada plugin.
4. Salve e verifique o resultado no blog publicado.
